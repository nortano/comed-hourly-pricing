# Release Notes - Hourly Pricing for ComEd

## [1.0.5] - (versionCode: 6)
### Improvements
- Tile and complication picker labels are now shorter: "5-Min Price" and "Hourly Price".
- Complications now display a dedicated bolt icon in the picker instead of the app icon.
- Tile picker previews now show accurate tile screenshots instead of the app icon.
- Deduplicated string resources and consolidated label strings.

## [1.0.1] - (versionCode: 2)
### Improvements
- Added native debug symbols to improve crash reporting and diagnostics in the Google Play Console.

## [1.0.0] - (versionCode: 1)
### Initial Release (en-US)
Initial release of Hourly Pricing for ComEd for Wear OS!

#### Features
- **Live Pricing**: View current 5-minute spot prices and hourly averages.
- **Color-Coded Tiers**: Quickly identify price levels (Free, Normal, Elevated, High) at a glance.
- **Wear OS Surfaces**: Access prices via the main app, watch face complications, or a dedicated home screen tile.
- **Background Sync**: Automatically stay updated with price refreshes every 15 minutes.
- **Offline Fallback**: Access last known prices when network is unavailable.
